Project Name: WISHBONE CONMAX IP Core

(See change Log at bottom of page for changes/updates)
 

Description

This is a WISHBONE Interconnect Matrix IP core.It can interconnect up to 8 Masters and 16 Slaves

Some of the main features are:

  • Up to 8 Masters
  • Up to 16 Slaves
  • 1, 2 or 4 priority levels
  • Fully configurable

Please see the spec for more details !
