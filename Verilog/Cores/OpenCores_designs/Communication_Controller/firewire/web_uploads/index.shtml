Project Name: FireWire

Description

FireWire, an Apple trademarked name for IEEE 1394 protocol, is a high performance serial bus protocol to provide easy-to-use, low-cost, high-speed communications. Click here for a good collection of links to IEEE 1394 documents.

The goal of the FireWire project is to provide free IEEE-1394 compliant Physical Layer and Link Layer IP cores. The FireWire project will also include firmware for Transaction Layer and Serial Bus Management Layer.

Current Status:

  • Researching FireWire protocol
  • Initial FireWire homepage was released on June 30, 2001.
