Instructions for running the testbench

Files:
- sim/gtkwave.sav (GTKWave waveform save file)
- sim/iverilog_gtkwave.scr (Bash script)
- sim/iverilog_gtkwave.cmd (Windows cmd script)

Requirements:
- Icarus Verilog simulator
- GTKWave waveform viewer (not essential)

Procedure:
1. First CD into the sim/ directory.
2. modify the test parameters in the script (the loop can be commented out)
3. run the script ./iverilog_gtkwave.scr
