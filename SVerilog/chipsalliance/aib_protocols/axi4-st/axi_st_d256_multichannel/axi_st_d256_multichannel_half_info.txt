// Data and Channel Size
  Logic Link Data Info
    LogicLink RSTRUCT  st       TX  257  RX    1
    RepStruct in Half Mode           x2       x2
                                -------  -------
    Total     RSTRUCT           TX  514  RX    2

  RSTRUCT Sub Channel Info
  Note: RSTRUCT describes the Replicated Struct on a Full rate channel.
        RSTRUCT will be replicated for Half rate per configuration and that is known as MAIN channel

    RSTRUCT: Each channel is Gen1Only PHY running at Full Rate with 40 bits
    RSTRUCT: 7x channels
    RSTRUCT: Total AIB bits is 280 bits

  MAIN Channel Info
    MAIN: Each channel is Gen1Only PHY running at Half Rate with 80 bits
    MAIN: 7x channels
    MAIN: Total AIB bits is 560 bits

       TX: No DBI
       TX: Strobe is Recoverable or non-existent
       TX: Marker is Recoverable or non-existent
       TX: Total RSTRUCT overhead bits across 7 Full Rate channels is 0
       TX: Total RSTRUCT data bits available 280
       TX: Total MAIN overhead bits across 7 Half channels is 0
       TX: Total MAIN data bits available 560

       RX: No DBI
       RX: Strobe is Recoverable or non-existent
       RX: Marker is Recoverable or non-existent
       RX: Total RSTRUCT overhead bits across 7 Full Rate channels is 0
       RX: Total RSTRUCT data bits available 280
       RX: Total MAIN overhead bits across 7 Half channels is 0
       RX: Total MAIN data bits available 560

  RSTRUCT TX needs  257 bits of data and has  280 bits available across 7x Gen1Only Half Rate channels so   23 spare bits
  RSTRUCT RX needs    1 bits of data and has  280 bits available across 7x Gen1Only Half Rate channels so  279 spare bits
  INFORMATION: At least one full channel unused for RX

  MAIN TX needs  514 bits of data and has  560 bits available across 7x Gen1Only Half Rate channels so   46 spare bits
  MAIN RX needs    2 bits of data and has  560 bits available across 7x Gen1Only Half Rate channels so  558 spare bits
